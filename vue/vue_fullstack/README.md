# Vue fullstack

## Usage

Add your supabase url and supabase api key in a `.env` file.

```env
VITE_SUPABASE_URL=XXX
VITE_SUPABASE_ANON_KEY=XXX
```

Then run the below.

```console
$ npm install @supabase/supabase-js
$ npm run serve
```
